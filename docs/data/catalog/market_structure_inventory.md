# Market-Structure Inventory

Tracked normalized files are under `Data/MarketStructure/`.

Primary public tables are `outputs/tables/T28` through `T37`.
Primary public figures are `outputs/figures/F30` through `F37`.

The extension is intentionally additive: cache-backed DefiLlama, Binance, and
CoinMarketCap pulls can enrich the tracked layer later, but the public repo
build succeeds without API keys.
