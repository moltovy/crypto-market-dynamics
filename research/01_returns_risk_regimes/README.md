# 01_returns_risk_regimes: Returns, Risk, and Regimes

## Research Question

How do BTC and ETH return distributions, volatility, drawdown, and tail behavior vary across transparent regimes?

## Current Findings

- ETH has n=2292 daily observations, annualized volatility=83.84%, max drawdown=-79.34%.
