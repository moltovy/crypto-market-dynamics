# Recruiter Summary

Crypto Market Factor Lab is a polished Python portfolio project showing data
engineering, quantitative research, crypto market knowledge, and disciplined
communication. It uses a frozen 2020-2026 BTC/ETH panel and turns it into
reports, figures, model cards, tests, and reproducible release packets.

The project is strongest for roles involving financial analytics, crypto
research, data engineering, quant development, and research tooling.
