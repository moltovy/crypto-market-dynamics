# Archive

This folder preserves development history and provenance that is not part of
the public project surface.

## Contents

- `manager_notes/`: manager-agent notes, prompts, workflows, and planning
  material.
- `legacy_portfolio_releases/`: historical generated release packets retained
  so canonical `outputs/` can be traced back to source artifacts.
- `showcase_drafts/`: earlier public-showcase drafts.
- `career_materials/`: job/career-oriented drafts intentionally removed from
  active public navigation.
- `legacy_specs/`: older paper and portfolio specs superseded by `docs/`.
- `old_reports/`: old drafts, notebooks, run summaries, and architecture
  inventories.
- `prior_ai_outputs/`: prior AI-generated research drafts retained for
  provenance.
- `release_process/`: PR, release, audit, and manual process documents.

Start with `README.md`, `docs/`, and `outputs/` for the current project.
